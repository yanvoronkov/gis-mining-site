# Страница отзывов

## Описание
Страница для отображения отзывов о компании из различных источников (Яндекс Карты, Google Карты, 2ГИС, Отзовик, Авито).

## Структура инфоблока

### Основные параметры
- **Тип инфоблока**: `content`
- **Символьный код**: `reviews`

### Поля элемента
- `NAME` — Имя автора отзыва
- `PREVIEW_TEXT` — Текст отзыва (с поддержкой HTML)
- `PREVIEW_PICTURE` — Скриншот отзыва

### Свойства элемента
1. `REVIEW_DATE` — Дата публикации отзыва (тип: Дата/время)
2. `SOURCE_URL` — Ссылка на источник отзыва (тип: Строка)

## Разделы инфоблока

Рекомендуемые символьные коды разделов для источников:

- `yandex_maps` — Яндекс Карты
- `google_maps` — Гугл Карты
- `maps_2gis` — 2ГИС
- `otzovik` — Отзовик
- `avito` — Авито

## Компонент

Используется стандартный компонент Битрикс `bitrix:news.list` с кастомным шаблоном `reviews`.

### Параметры компонента
- Сортировка по `PROPERTY_REVIEW_DATE` по убыванию
- Количество элементов на странице: 20
- Фильтрация по разделам через GET-параметр `SECTION_CODE`

## Файлы

### Основные файлы
- `/reviews/index.php` — Страница отзывов
- `/local/templates/main/components/bitrix/news.list/reviews/template.php` — Шаблон компонента
- `/local/templates/main/components/bitrix/news.list/reviews/style.css` — Стили
- `/local/templates/main/components/bitrix/news.list/reviews/.parameters.php` — Параметры шаблона

## Особенности

1. **Вкладки с фильтрацией**: Автоматическое переключение между разделами с активной подсветкой
2. **SEO-оптимизация**: Полный набор мета-тегов и Open Graph
3. **Адаптивность**: Отзывчивый дизайн для мобильных устройств
4. **Кэширование**: Включено с временем 36000000 секунд
5. **Пагинация**: Ручная пагинация (идентична странице каталога ASIC)

## URL структура

- Главная страница: `/reviews/`
- Фильтр по источнику: `/reviews/?SECTION_CODE=yandex_maps`
- Пагинация: `/reviews/?SECTION_CODE=yandex_maps&PAGEN_1=2`

## Добавление нового источника

1. Создайте новый раздел в инфоблоке `reviews`
2. Укажите символьный код (латиницей)
3. Добавьте иконку источника в поле `PICTURE` (опционально)
4. Вкладка автоматически появится на странице
