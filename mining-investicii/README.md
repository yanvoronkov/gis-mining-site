# GIS Mining - Сайт компании по майнингу криптовалюты

## Описание проекта
Адаптивный одностраничный сайт для компании GIS Mining, предлагающей услуги по майнингу криптовалюты. Сайт разработан с использованием принципа "Mobile First".

## Структура проекта
- `index.html` - главная страница сайта
- `styles.css` - стили сайта
- `scripts/` - JavaScript функционал
- `images/` - директория для изображений
- `fonts/` - директория для шрифтов
- `send_lead.php` - скрипт для отправки данных из контактной формы в CRM Битрикс

## Особенности
- Адаптивный дизайн (Mobile First)
- Интерактивные элементы: аккордеон, анимация статистики
- Валидация формы обратной связи
- Оптимизированная производительность
