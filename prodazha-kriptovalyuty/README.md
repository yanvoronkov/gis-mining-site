# Страница "Продажа криптовалюты"

## Описание
Страница легальной продажи криптовалюты с комиссией 1,5%. Включает информацию о процессе работы, преимуществах и форму для расчета сделки.

## URL
`/prodazha-kriptovalyuty/`

## Структура страницы

### 1. Hero Section (Главный экран)
- Заголовок с акцентом на легальности и комиссии 1,5%
- Адаптивное изображение (десктоп/мобайл)
- Список преимуществ с иконками (4 пункта):
  - Полный комплект документов по требованиям ФНС
  - Моментальное зачисление на расчетный счет
  - Фиксация курса и суммы до начала сделки
  - Порог входа от 300 000 руб.

### 2. Process Section (Процесс работы)
- Заголовок "Как работаем?"
- 4 шага работы в виде карточек:
  1. Присылаете объем - получаете расчет
  2. Заключаем Договор
  3. Фиксируем условия по Спецификации
  4. Отправляете криптовалюты и получаете рубли + документы

### 3. CTA Section (Форма обратной связи)
- Адаптивное изображение
- Форма "Рассчитайте сумму сделки" с полями:
  - Телефон (обязательное)
  - Email (необязательное)
  - Политика конфиденциальности
  - Кнопка отправки

## Интеграция формы

### Обработчик
Форма использует существующий обработчик `/send_lead.php` через AJAX

### Параметры формы
- **form_name**: "Продажа криптовалюты"
- **data-metric-goal**: "crypto-sale-lead" (для отслеживания конверсий в метрике)
- **source_id**: "WEB"

### Поля формы
- `client_phone` - телефон (обязательное, с маской)
- `client_email` - email (необязательное)
- UTM-метки (скрытые поля, автозаполнение через JS)
- `page_url` (скрытое поле, автозаполнение через JS)

### JS-классы
- `.js-ajax-form` - для AJAX-отправки формы
- `.js-phone-mask` - для маски телефона
- `.form-input` - стандартизация стилей полей
