# Инструкция по добавлению страницы в меню сайта

## Вариант 1: Добавление в главное меню

Если вы хотите добавить страницу "Продажа криптовалюты" в главное меню сайта, отредактируйте файл:
`.top.menu.php`

Добавьте новый пункт меню:

```php
Array(
    "Продажа криптовалюты", 
    "/prodazha-kriptovalyuty/", 
    Array(), 
    Array(), 
    "" 
),
```

### Пример добавления после пункта "Строительство":

```php
Array(
    "Строительство", 
    "/newstroitelstvo/", 
    Array(), 
    Array(), 
    "" 
),
Array(
    "Продажа криптовалюты", 
    "/prodazha-kriptovalyuty/", 
    Array(), 
    Array(), 
    "" 
),
Array(
    "Оплата и доставка", 
    "/oplata-i-dostavka/", 
    Array(), 
    Array(), 
    "" 
),
```

## Вариант 2: Добавление в раздел "Другие услуги" (мегаменю)

Если вы хотите добавить страницу в выпадающее меню "Другие услуги", вам нужно:

1. Открыть файл с настройками мегаменю (обычно это компонент Битрикс или отдельный файл конфигурации)
2. Добавить новую карточку с ссылкой на `/prodazha-kriptovalyuty/`

### Пример структуры карточки для мегаменю:

```html
<a href="/prodazha-kriptovalyuty/" class="megamenu__card">
    <div class="megamenu__card-title">Продажа криптовалюты</div>
    <div class="megamenu__card-description">
        Легальный обмен криптовалюты с комиссией 1,5%
    </div>
    <div class="megamenu__card-arrow">
        <svg><!-- иконка стрелки --></svg>
    </div>
</a>
```

## Вариант 3: Добавление в футер

Чтобы добавить ссылку в футер сайта, отредактируйте шаблон футера:
`/local/templates/main/footer.php`

Найдите блок навигации футера и добавьте новую ссылку:

```html
<li class="footer__nav-item">
    <a href="/prodazha-kriptovalyuty/" class="footer__nav-link">
        Продажа криптовалюты
    </a>
</li>
```

## Важные замечания

### SEO
- При добавлении ссылок используйте осмысленный anchor text (текст ссылки)
- Не используйте "Читать далее" или "Подробнее" без контекста
- Желательно использовать ключевые слова: "продажа криптовалюты", "обмен криптовалюты"

### Аналитика
Рекомендуется добавить отслеживание кликов на ссылки:

```html
<a href="/prodazha-kriptovalyuty/" 
   onclick="ym(XXXXXX,'reachGoal','crypto-sale-link-click')">
    Продажа криптовалюты
</a>
```

### Внутренняя перелинковка
Рекомендуется добавить ссылки на страницу "Продажа криптовалюты" со следующих страниц:
- Главная страница
- Страница "О компании"
- Страница "Контакты"
- Страница "Каталог" (если применимо)
- Блог (в релевантных статьях)

## Проверка после добавления

После добавления страницы в меню обязательно проверьте:
- [ ] Ссылка работает корректно
- [ ] Текст ссылки читабелен и понятен
- [ ] Активное состояние ссылки работает (если страница открыта)
- [ ] Мобильная версия меню отображается корректно
- [ ] Нет дублирующих ссылок в меню
- [ ] Порядок пунктов меню логичен
